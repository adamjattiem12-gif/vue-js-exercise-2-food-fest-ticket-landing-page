# food-fest-ticket-landing-page

Cape Town Food Fest Ticket Landing is a Vue 3 single-page landing page for an
outdoor food festival. It displays Bronze, Silver, and Gold ticket tiers with
prices, descriptions, benefits, a featured tier highlight, and an interactive
favourite button.

## Features

- Dynamic ticket cards rendered from local data
- Reusable `TicketCard` component
- Featured Silver tier styling
- Favourite toggle with a live favourite count
- Responsive layout for mobile and desktop screens

## Project Setup

Install dependencies:

```sh
npm install
```

Run the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```
