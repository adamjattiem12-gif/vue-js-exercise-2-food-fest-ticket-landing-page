# food-fest-ticket-landing-page

Cape Town Food Fest Ticket Landing is a Vue 3 single-page landing page created for an annual outdoor food festival in Cape Town. The page showcases three ticket tiers: Bronze, Silver, and Gold. Visitors can compare prices, view benefits, see the featured ticket tier, and favourite a ticket option to simulate an interactive ticketing experience.

## Project Overview

This project was built as part of Module 1 - Frontend Web Development, Exercise 02. The goal was to create a modern, responsive Vue.js landing page that displays ticket options using reusable components and locally stored data.

## Features

- Vue 3 single-page application
- Dynamic ticket tier rendering from local data
- Reusable ticket card component
- Bronze, Silver, and Gold ticket options
- Featured ticket tier highlight
- Favourite button interaction
- Live favourite count
- Responsive layout for mobile and desktop screens
- Clean event-style landing page design

## Technologies Used

- Vue 3
- Vite
- HTML
- CSS
- JavaScript
